  1. How to insert data in particular database ?
    db.collectionname.insertOne({ name:'m 40', brand: 'samsung', price: 18000 });
  2. How to insert data from Mongodb UI ?
    Go to your database,click on collection,click on Add Data button then insert data init in object form then click on Insert button to insert.
  3. How to see your inserted data or existing data via terminal ?
    db.collectionname.find() then your all existing record show in terminal.
  4. How to update data from UI ?
    Simply hover on data then click on pencil icon after that you have option to edit you edit and click on Update button.
  5. How to update data from Terminal ?
    db.collectionname.updateOne({"name":"y10"},{$set:{'brand':'Nokia'}})
  6. How to delete data from UI ?
    Simply hover on record there is one delete option,simply click on it for delete that particular record.
  7. How to delete data from Terminal ?
    db.collectionname.deleteOne({name:"y11"})
